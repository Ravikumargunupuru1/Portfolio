I'm creative developer
Hi, I'm Ravi Kumar Gunupuru, a final-year B.Tech student specializing in Computer Science and Engineering at Krishna University. With a strong foundation in programming and a passion for innovation, I am dedicated to leveraging technology to create impactful solutions.
Currently, I am working on my final-year project titled "Advanced Personal AI Voice Assistant", which focuses on developing an intelligent voice assistant capable of performing diverse system tasks, similar to Google Assistant and Alexa. This project stands out by incorporating advanced features such as human-like speech patterns and sentiment-based responses using Sentiment Analysis, AI & ML, and Natural Language Processing (NLP) techniques. It reflects my commitment to pushing the boundaries of AI to enhance user experiences.
I am excited to further explore the fields of AI and Machine Learning, applying my skills to real-world challenges and making meaningful contributions to the tech industry.

My Portfolio Some Of My Works

Business contract Validation

The application processes contract documents uploaded in PDF format, leveraging advanced natural language processing (NLP) techniques to clean and normalize text. It identifies entities and clauses, compares the contract with selected templates, highlights deviations, and provides insightful summaries.

Django School Management

Django School Management is a web-based application designed to streamline and simplify the management of educational institutions. Built using the Django framework, it allows administrators to efficiently manage student records, faculty details, attendance, grades, and schedules. The system also supports features like class management, fee tracking, and communication tools between students, teachers, and parents. With its user-friendly interface, Django School Management enhances the overall administrative experience, making it easy to manage school operations in a centralized, secure platform.

Multiple Disease Prediction using Streamlit

The Multiple-Disease-Prediction-Streamlit model is a web-based application built using Streamlit that leverages machine learning algorithms to predict the likelihood of three major health conditions: Parkinson's disease, diabetes, and heart disease.

RK Text Editor

RK Text Editor is a simple and user-friendly text editor built using Python's Tkinter library. It allows users to open, edit, and save text files with ease. The project includes a modern GUI design with dynamic buttons that change color when hovered over, enhancing the user experience.

Advanced calculator

A simple, user-friendly calculator application built using Python's Tkinter library. This calculator supports basic arithmetic operations, features a graphical user interface (GUI), and provides error handling for invalid input and zero division.

Ibomma-clone

This is a clone of the iBomma website, built for educational purposes only. iBomma is a popular online streaming platform where users can watch and download Telugu movies. This project replicates its frontend and backend structure with the goal of understanding how similar sites function. Please note that this project does NOT host any real copyrighted content, and is strictly for learning purposes.
